# NavigationInRoom

Небольшой Unity-прототип, в котором демонстрируется движение ИИ по NavMesh в 2D-сцене:  
мы выбираем комнату, а агент прокладывает путь и идёт туда.

## О проекте

Сцена — это набор комнат, соединённых проходами.  
По полу «лежит» NavMesh, по которому двигается AI-агент.

Основная логика:

- есть несколько **комнат-зон**;
- игрок выбирает целевую комнату (через UI / кликом по объекту);
- **NavMeshAgent** получает новую точку назначения;
- ИИ автоматически находит путь и доходит до выбранной комнаты, обходя стены и препятствия.

Задача прототипа — показать:

- базовую настройку NavMesh в 2D-проекте;
- управление целями для ИИ;
- простую и наглядную демонстрацию навигации между комнатами.
